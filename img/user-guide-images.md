## Recommended Shooting Priorities

Top 10 screenshots/recordings to cover the maximum HIGH-priority items in one NinjaTrader session.

### 1. Annotated Footprint Bar Anatomy (covers #7, #8, #9, #28, #30)
**Setup:** ES 5-min chart, Tick Replay ON, mzFootprint with BidAsk style, Saturation color mode, Imbalance 200%, POC + VA enabled.
**What to capture:** One footprint bar zoomed in showing: bid/ask columns, delta values, diagonal imbalance highlighted cells (green = buy, red = sell), POC marker (yellow), VA shading. Annotate with arrows in an image editor afterward.
**Reuse:** Crop/annotate for concepts/order-flow.md anatomy, imbalance explanation, and indicators/mzFootprint.md styles.

### 2. Imbalance S/R Zones + Absorption Zones (covers #10, #31, #32)
**Setup:** Same ES 5-min chart. Enable S/R zones: consecutive levels = 3, volume filter = 10. Enable Absorption #1: 100%, depth 2, S/R zones enabled.
**What to capture:** Several bars showing green support zones and red resistance zones projected forward. At least one absorption zone visible at a bar extreme.
**Reuse:** Concepts and indicator pages.

### 3. Volume Profile Full Anatomy + Level Modes (covers #11, #14, #15, #40, #41, #43, #44)
**Setup:** ES 30-min chart, mzVolumeProfile, Session mode, Ladders view, VWAP Dynamic + StdDev1 + StdDev2 enabled, POC Extended, VAH/VAL Naked.
**What to capture:** Full session profile with POC line, VAH/VAL lines, VWAP + bands, HVN/LVN visible. Take additional captures: switch view to Area, then Contour. Switch type to BuySell, Delta, VolumeDelta. Change POC mode through On/Extended/Naked/Developing.
**Reuse:** Concepts volume-profiling anatomy, level modes comparison, profile views/types comparison, VWAP bands.

### 4. TPO Profile with Structure (covers #17, #18, #19, #20, #45)
**Setup:** ES 30-min chart, mzVolumeProfile with Chart profiles = TPO, Letter mode, 30-min period, Rotate colors ON, IB 60 min, VA box ON, Halfback ON.
**What to capture:** Complete TPO profile showing letters, POC row highlighted, VA box, IB range, open/close markers, single prints. Then switch to different sessions to capture Normal (bell), b-shape, P-shape examples.
**Reuse:** Concepts market-profile TPO anatomy, day types, IB range, and indicator TPO section.

### 5. mzVolumeDelta Display Modes + Alignment (covers #54, #55)
**Setup:** ES 5-min chart, mzVolumeDelta. Capture in Volume mode: switch Align through Total, Stacked, Opposite. Then capture in Delta mode: switch Show through Histogram, Candles, Cumulative.
**What to capture:** 6 screenshots (3 alignment, 3 delta display). Arrange as side-by-side comparisons.

### 6. mzBigTrade Markers + DOM Pressure (covers #58, #59, #60, #66)
**Setup:** ES 1000-tick chart with live/replay data. mzBigTrade Manual filter, min volume 50, Iceberg ON, Show DOM pressure ON. Bubble marker.
**What to capture:** First in Default mode — bubbles on bars with DOM pressure triangles (filled and outline). Switch marker to Line, then Box, then Bar for comparison. Switch to Tape mode for Default vs Tape comparison. Enable Trades Volume Profile for iceberg use case (Marker size = Iceberg, Heatmap color).
**Reuse:** Marker types comparison, Default vs Tape, DOM pressure, iceberg use case.

### 7. mzMarketDepth Full Setup (covers #68, #69, #71, #75)
**Setup:** ES 5-min chart with live/replay Level 2 data. mzMarketDepth: Show histogram ON, Heatmap color, Code extremal ON, Historical DOM ON with AdaptiveLess filtering. Liquidity Migration ON: Added filter 50, Removed filter 50, Overall OfferBidDelta candles. Liquidity Overall OfferBid line at top.
**What to capture:** Full chart showing: right margin DOM histogram with imbalanced levels, historical heatmap on past bars with extreme levels, migration markers on chart, liquidity candles at bottom, liquidity lines at top. Capture the whole chart first (comprehensive), then zoom into sections.

### 8. Delta Divergence Signals (covers #22, #23, #76, #77, #78)
**Setup:** ES 1000-tick chart, mzDeltaDivergence, ZigZag deviation 2, Price deviation min 4, Delta deviation min 150. Cumulative delta display ON.
**What to capture:** Chart section showing at least one LONG (green zone, price new low with positive delta) and one SHORT (red zone, price new high with negative delta) divergence. Breakpoint arrows visible. Statistics labels showing price/delta values. Cumulative delta candles in lower panel.
**Reuse:** Concepts delta-divergence, indicator LONG/SHORT examples, cumulative delta with divergence use case.

### 9. Strategy Framework Visualization (covers #81, #82, #83, #86, #87)
**Setup:** ES 5-min chart with FootprintAction strategy in Manual mode. Enable Pattern dashboard ON, Pattern background ON. Also load GhostResistance in Manual mode on a separate chart.
**What to capture:** FootprintAction: pattern dashboard overlay showing signal states, colored pattern backgrounds on validated signals. GhostResistance: a reversal setup with price spike beyond a level, absorption, and reversal. Capture Control Panel if visible.
**Reuse:** Strategy framework pages and built-in strategies.

### 10. Activation Window + Tick Replay Setting (covers #5, #89)
**Setup:** Fresh NinjaTrader with MZpack not yet activated, or a chart with Data Series dialog open.
**What to capture:** (a) Activation Window showing license key field, Activate and START TRIAL buttons. (b) Data Series dialog with Tick Replay checkbox highlighted.
**Reuse:** Licensing activation, troubleshooting tick replay.
