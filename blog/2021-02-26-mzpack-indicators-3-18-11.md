## What's New

### mzBigTrade

- Added Trades Volume Profile containing all trades displayed on the chart
- This feature works best when paired with the "Marker size relative to" parameter configured for Iceberg/DOM pressure/DOM support, enabling traders to identify significant price levels
- Removed Presentation -- Min shape extent/Min line length parameters for improved and proportional trades visualization

### Common

- Added option "General -- Small toolbar buttons"
